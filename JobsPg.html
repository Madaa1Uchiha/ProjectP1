Job Descriptions

Front End Web Developer/Back End Web Developer

We are looking for a Front End Web Developer/Back End Web Developer to join our team. The ideal candidate will have experience in HTML, CSS, JavaScript, and server-side programming.

Responsibilities include:

  • Develop and maintain web applications
  • Collaborate with designers and other developers
  • Optimize applications for maximum speed and scalability
  • Ensure the technical feasibility of UI/UX designs
  • Participate in code reviews

Qualifications:

  • Proven experience as a Front End Web Developer/Back End Web Developer(Projects, previous employments)
  • Strong knowledge of HTML, CSS, JavaScript, and server-side programming
  • Experience with responsive design and cross-browser compatibility
  • Familiarity with version control systems (e.g., Git)
  • Excellent problem-solving skills

Fullstack Web Dev(Leadership)

We are looking for a Fullstack Web Developer to join our team. The ideal candidate will have experience in both front-end and back-end development.

Responsibilities include:

  • Many of the same as Front End Web Developer/Back End Web Developer
  • Be able to lead or liase with Front and Back end developers
  • Be able to work with the team to develop and maintain web applications
  • Collaborate with designers and other developers
  • Optimize applications for maximum speed and scalability
  • Help Juniour developers

Qualifications:

  • Proven experience as a Fullstack Web Developer
  • Strong knowledge of HTML, CSS, JavaScript, and server-side programming
  • Experience with responsive design and cross-browser compatibility
  • Familiarity with version control systems (e.g., Git)
  • Excellent problem-solving skills
  • Experience with leadership or mentoring

Data Scientist

We are looking for a Data Scientist to join our team. The ideal candidate will have experience in data analysis, machine learning, and statistical modeling.

Responsibilities include:

  • Analyze large datasets to extract insights and trends
  • Develop and implement machine learning models
  • Collaborate with cross-functional teams to understand data needs
  • Communicate findings to stakeholders
  • Stay up-to-date with the latest data science trends and technologies

Qualifications:

  • Proven experience as a Data Scientist
  • Strong knowledge of data analysis, machine learning, and statistical modeling
  • Experience with programming languages such as Python or R
  • Familiarity with data visualization tools (e.g., Tableau, Power BI)
  • Excellent problem-solving skills
  • Strong communication skills
